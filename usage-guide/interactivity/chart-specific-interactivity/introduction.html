FusionCharts Suite XT supports charts that allow user-interactivity. This allows the user to improve viewing by rendering only as much data as fits on the screen, modify existing data on the chart to understand its impact, and view only specific data.

The interactivity features supported by FusionCharts Suite XT inlcude:

  • Scrolling through a chart

  • Dragging chart elements

  • Legends

Scrolling Through a Chart

FusionCharts Suite XT includes scrollable charts - charts that allow you to plot large quantities of data on one chart and then scroll through to view only as much data as required.

To understand how the scrolling interactivity works, click here.

Dragging Chart Elements

FusionCharts Suite XT includes drag-able charts that allow you to visually manipulate data on a chart, submit it, and observe its impact.

To understand how the dragging interactivity works, click here.

Legends

A legend is a chart element that displays the series name of each dataset in charts that plot data for multiple datasets. It helps to correlate a data plot to its series name with the color used to represent the data belonging to that series.
